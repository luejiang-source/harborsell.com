# Deploy Your Tech Store in 5 Minutes

## Step 1: Create GitHub Repository (2 minutes)

1. Go to **github.com** and sign up (free)
2. Click **"New"** to create a new repository
3. Name it: `techhub`
4. Choose **Public**
5. Click **"Create repository"**

## Step 2: Upload Files (1 minute)

1. Click **"Add file"** → **"Upload files"**
2. Drag and drop `index.html` into the upload area
3. Click **"Commit changes"**

## Step 3: Enable GitHub Pages (1 minute)

1. Click **Settings** tab
2. Scroll to **"Pages"** section
3. Under "Source", select **"main"** branch
4. Click **"Save"**

## Step 4: Get Your URL (1 minute)

Your site is now live at:
```
https://YOUR-USERNAME.github.io/techhub
```

✅ **You're done!** Share this link with anyone.

---

## Faster Option: Deploy with Netlify (2 minutes)

1. Go to **netlify.com**
2. Click **"New site from Git"**
3. Select GitHub, authorize, and choose your `techhub` repo
4. Click **"Deploy site"**
5. Your site is live at `random-name.netlify.app`

---

## Custom Domain (Optional)

Want `mytechstore.com` instead?

1. Buy domain from Namecheap ($8-15/year)
2. In GitHub Pages/Netlify settings, add custom domain
3. Point nameservers to GitHub/Netlify
4. Done!
