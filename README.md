# TechHub - Technology Store

A professional static e-commerce website for selling technology products.

## Features

✅ **Responsive Design** - Works on mobile, tablet, and desktop
✅ **Professional UI** - Built with TailwindCSS
✅ **Product Showcase** - 6 featured tech products
✅ **Contact Form** - Get customer inquiries
✅ **Fast Loading** - Pure HTML/CSS, no backend needed
✅ **SEO Ready** - Proper meta tags and structure
✅ **Free Hosting** - Deploy to GitHub Pages instantly

## Files

- `index.html` - Main website (all content in one file)

## Customization

### Change Products
Edit the product cards in `index.html`:
```html
<h3 class="text-2xl font-bold mb-2">Your Product Name</h3>
<p class="text-gray-600 mb-4">Your product description</p>
<span class="text-3xl font-bold text-purple-600">$Your Price</span>
```

### Change Colors
Replace `purple-600` with your preferred color:
- `blue-600` for blue
- `green-600` for green
- `red-600` for red
- `indigo-600` for indigo
- etc.

### Change Company Info
Update footer section with your actual contact details:
```html
<p><strong>Email:</strong> your-email@example.com</p>
<p><strong>Phone:</strong> Your phone number</p>
```

## Deployment Options

### Option 1: GitHub Pages (Free, Easiest)

1. Create GitHub account (free at github.com)
2. Create new repository named `techhub`
3. Upload `index.html` to the repository
4. Go to Settings → Pages → Select "main" branch
5. Your site is live at `username.github.io/techhub`

### Option 2: Netlify (Free)

1. Go to netlify.com
2. Click "New site from Git"
3. Connect your GitHub repository
4. Deploy (automatic after each update)
5. Get a free `.netlify.app` domain

### Option 3: Vercel (Free)

1. Go to vercel.com
2. Click "New Project"
3. Import from GitHub
4. Deploy
5. Get a free `.vercel.app` domain

### Option 4: Custom Domain

Buy domain from Namecheap, GoDaddy, or Google Domains
Point to your GitHub Pages/Netlify/Vercel deployment
